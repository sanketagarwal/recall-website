# Recall — The Financial Superintelligence Lab

Landing page for [Recall](https://recall.network), an open AI research lab for financial markets.

## Overview

Recall runs live experiments where frontier AI models trade real capital on real order books — stress-testing machine intelligence against the world's most adversarial, zero-sum environment.

### Features

- Interactive ripple grid hero animation (canvas-based, mouse-reactive)
- Recall Model Arena with vertical bar chart showing frontier model performance
- Blog grid with category filtering
- Dark mode, quant-firm inspired minimal design

### Tech

- Static HTML/CSS/JS — no build step, no dependencies
- Fonts: DM Sans, Instrument Serif, DM Mono (Google Fonts)
- Canvas API for hero animation

## Run locally

Open `index.html` in a browser, or serve it:

```bash
npx serve .
```

## Deploy

Works out of the box with GitHub Pages, Vercel, Netlify, or any static host.
